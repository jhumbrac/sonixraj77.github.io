Assignment 2- Creating a mock-up of a multi-page signup and checkout flow

This assignment is about creating a responsive website for a vegetable delivering service, to the homes of people living in the food desert areas, where access to fresh produce is difficult. These areas usually have a corner store which hardly carries fresh produce. They carry packaged food and frozen meats. Here is a website attempting to make buying fresh produce easy, user friendly and quick. The objective is helping the people order online and have produce delivered to their doorstep. In designing the flow of the website I followed the following steps

  1. Learn
  2. Plan
  3. Design
  4. Code

Step 1. Learn

To understand and learn more about the user a questionair was created based on which the various elements of the website would be created

Step 3. Design

Wireframes
Sketches helped me decide everything that needed to go into a website and the flow of things. Wireframes helped decide the hierarchy of elements and what would go where. It was very important that the whole website be very easy to navigate and does not take too much time to load with few images. Inserting an element that showed the user the steps being followed was critical in the feel of the website.

Following the above pattern I have created a website which has the following pages

  1. Landing page
  2. Selection Page
  3. Signup Page
  4. Create account page
  5. Confirm and checkout
  6. Thankyou and confirmation page

The objective of the assignment was to start designing from the signup/subscribe page however in understanding the flow of things and for the user to get inspired to embrace this almost new way of eating I started my first page with selecting a box of colorful produce and then signing up and proceeding to checkout.
